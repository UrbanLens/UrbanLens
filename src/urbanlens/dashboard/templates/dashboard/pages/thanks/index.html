Credits & acknowledgements

Standing on the shoulders
of open data & open source.

UrbanLens is built from community-maintained maps, geospatial datasets, and free software. This page lists the projects and providers whose work makes the platform possible.

Map tiles & geospatial data

What you see on the map

Base layers and overlays on the main map draw from the sources below. Attribution shown will vary by layer; the map footer shows credits for whichever layers are active.

map

OpenStreetMap

Community-maintained street map data used for the default street base layer and many location features. © OpenStreetMap contributors, available under the Open Database License (ODbL).

terrain

OpenTopoMap

Topographic hillshade and contour tiles built from SRTM elevation data and OpenStreetMap. Map style © OpenTopoMap (CC-BY-SA); elevation data © SRTM.

satellite_alt

Esri

Satellite imagery and geopolitical boundary reference layers. Tiles © Esri — sources include Esri, i-cubed, USDA, USGS, AEX, GeoEye, Getmapping, Aerogrid, IGN, IGP, UPR-EGP, and the GIS User Community.

dark_mode

CARTO

Dark Matter basemap tiles used when map dark mode is active on street and terrain views. © OpenStreetMap contributors © CARTO.

cloud

OpenWeatherMap

Precipitation and cloud overlay tiles on the optional weather layer. Map data © OpenWeatherMap, subject to their terms of use.

public

Nominatim / OpenStreetMap

Reverse geocoding and address lookup via the Nominatim service, operated by OpenStreetMap contributors under the ODbL.
