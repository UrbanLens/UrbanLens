Google Takeout / Google My Maps

We support Google Maps lists, Google My Maps, and Google Earth.

  1. Go to Google Takeout and select the Maps data you want to export.
  2. Click "Deselect All"
  3. Check any of the following sections you want to import:
    1. Google Earth
    2. Maps
    3. Maps (your places)
    4. My Maps
    5. Saved
    6. Street View
    7. Timeline
  4. Click "Next Step"
  5. Google will email you when the download is ready.
  6. Upload the exported file to UrbanLens.
Tip: You will have a chance to choose which pins to import.

Multiplottr

  1. Login to Multiplottr.
  2. Click "Map Settings"
  3. Click "Download KML File"
  4. Upload the exported file to UrbanLens.
    5. Tip: You will have a chance to choose which pins to import.

GIS portals and open data

City, county, state, federal, university, and preservation datasets often export Shapefile, GeoJSON, CSV, or ZIP archives.

  1. Find the dataset's download or export panel.
  2. Prefer GeoJSON for point datasets. Use Shapefile or ZIP when GeoJSON is not offered.
  3. For CSV exports, make sure latitude and longitude columns are included.

OpenStreetMap / Overpass Turbo

  1. Build a query in Overpass Turbo.
  2. Export the results as GeoJSON for the cleanest import, or OSM XML if you need original OSM tags.
  3. Review imported pins carefully. Public map tags can be incomplete or inaccurate.
