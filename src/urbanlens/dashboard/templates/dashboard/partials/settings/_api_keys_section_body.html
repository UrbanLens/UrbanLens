Let another application create pins on your behalf. A key can only read your account's UUID and create pins - nothing else.

How to use an API key

Send requests with an Authorization: Bearer <key> header.

  • GET {{ external_api_whoami_url }} - returns {"uuid": "..."}
  • POST {{ external_api_pins_url }} - create a pin. Body: {"name": "...", "latitude": 42.5, "longitude": -73.5} (or "address" instead of latitude/longitude) 
curl -X POST {{ external_api_pins_url }} \
  -H "Authorization: Bearer ulk_..." \
  -H "Content-Type: application/json" \
  -d '{"name": "Old Mill", "latitude": 42.5, "longitude": -73.5}'
