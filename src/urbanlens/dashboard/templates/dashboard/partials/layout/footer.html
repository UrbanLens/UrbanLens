Any page with its own embedded Leaflet map should disable Leaflet's on-map attribution control (attributionControl: false) - the required attribution text belongs here instead of floating over the map. Two ways to opt in: - Pass `map_attribution` (a plain string) for a map whose layer never changes after the initial render (e.g. the safety check-in route map). - Pass `show_map_footer=True` (with `map_attribution` omitted/blank) for a map whose attribution changes at runtime (switching base layers/overlays) - JS then keeps `#page-footer-attribution-text` (a stable, page-wide id) up to date via MapLayers' `onAttribution` callback, the same pattern the main map page uses. {% endcomment %}
