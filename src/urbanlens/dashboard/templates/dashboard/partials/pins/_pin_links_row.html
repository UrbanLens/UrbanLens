{% comment %} Compact "Links" row for the Details section (pin_overview_partial.html) and the Location wiki page - a list of external website links with an inline add form, styled like the other detail-item rows (Official Name, Address, ...). Usage: rendered via PinLinksView/LocationLinksView, which supply `links`, `add_url`, `delete_url_name`, `row_id`, `owner_slug` (the url's first argument), and either `pin` or `wiki` for display purposes. hide_when_empty: when true and there are no links, the whole row (including the "No links yet." placeholder) stays hidden - the caller is expected to provide its own "Add a link" icon button elsewhere (e.g. in the section header) that opens the shared #pin-link-add-dialog. In this mode the row's own inline add-toggle/form is omitted entirely - both "add the first link" and "add another" go through that one dialog, so there's no separate inline control whose Cancel button needs to hide the row back down again. See pin_overview_partial.html's Details card header for the pin page's version. Unset/false (the wiki page's usage) keeps the original always-visible row with its own inline add-toggle/form. {% endcomment %}
